# 商品到期提醒系统

一个简单实用的商品到期提醒应用，帮助您管理商品的保质期信息。

## 功能特点

### 1. 商品管理
- 支持EAN-13/8条码输入（适合扫码枪使用）
- 自动根据条码带出商品名称（需提前设置映射）
- 支持手动输入或通过生产日期+保质期自动计算到期日期
- 商品列表展示，包含条码、名称、生产日期、保质期、到期日期等信息

### 2. 条码映射
- 建立商品条码与商品名称的映射关系
- 输入条码时自动带出对应商品名称
- 编辑映射时自动同步更新商品列表中的商品名称
- 支持映射的添加、编辑和删除

### 3. 到期状态管理
- 自动计算商品到期状态：
  - 正常：3个月以上
  - 1-3个月：即将到期预警
  - 1个月内：紧急提醒
  - 已过期：过期商品

### 4. 数据可视化
- 使用柱状图直观展示不同到期状态的商品数量分布
- 实时更新图表数据

### 5. 数据管理
- **导出CSV**：将商品数据导出为CSV文件
- **导入CSV**：从CSV文件导入商品数据
- **清空数据**：一键清空所有商品和映射数据

### 6. 数据同步
- 集成LeanCloud实现数据云同步功能
- 需要配置LeanCloud账号信息

## 使用方法

### 1. 基本使用
1. 打开 `index.html` 文件
2. 在商品条码输入框中输入或扫描条码
3. 选择生产日期和保质期，系统会自动计算到期日期
4. 点击"添加商品"按钮保存

### 2. 配置条码映射
1. 在"条码-名称映射管理"区域
2. 输入商品条码和对应的商品名称
3. 点击"添加映射"按钮保存
4. 之后输入该条码时，商品名称会自动填充

### 3. 数据导入导出
- **导出**：点击"导出CSV"按钮，系统会生成包含所有商品数据的CSV文件
- **导入**：点击"导入CSV"按钮，选择要导入的CSV文件

### 4. 配置LeanCloud数据同步
1. 注册LeanCloud账号并创建应用
2. 在 `script.js` 文件中配置：
   ```javascript
   const APP_ID = 'your_app_id'; // 替换为你的App ID
   const APP_KEY = 'your_app_key'; // 替换为你的App Key
   AV.init({
       appId: APP_ID,
       appKey: APP_KEY,
       serverURL: 'https://your_server_url.leancloud.cn' // 替换为你的服务器地址
   });
   ```
3. 点击"同步数据"按钮进行数据同步

- **前端框架**：纯HTML/CSS/JavaScript，无框架依赖
- **图表库**：Chart.js
- **云存储**：LeanCloud（可选）
- **本地存储**：localStorage

## 注意事项

1. 请确保在现代浏览器中使用（推荐Chrome、Firefox、Edge等）
2. 条码映射功能需要提前设置才能自动带出商品名称
3. 首次使用时，需要配置LeanCloud信息才能使用数据同步功能
4. 导入CSV文件时，请确保格式与导出的CSV文件格式一致

## 故障排除

### 条码无法自动带出商品名称
- 检查条码是否已添加到映射列表
- 确保条码输入正确，无多余空格

### 数据同步失败
- 检查LeanCloud App ID和App Key是否配置正确
- 确保网络连接正常
- 检查LeanCloud控制台中的应用状态

### 图表不显示
- 确保浏览器支持Canvas元素
- 检查是否有JavaScript错误（可通过浏览器开发者工具查看）
